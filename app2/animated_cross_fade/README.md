# animated_cross_fade
## 简介
AnimationCrossFade能够让我们在两个不同的Widget中进行淡化过渡

**它需要传递四个参数：**
- Duration：动画过渡时间
- firstChild：第一个child Widget
- SecondChild：第二个child Widget
- crossFadeState：用于切换当前显示的是第一个childWidget还是第二个childWidget
